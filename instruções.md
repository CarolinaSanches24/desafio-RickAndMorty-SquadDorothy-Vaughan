**Para criar o ambiente Virtual**

```shell
py -3 -m venv .venv
```

**Para Ativar o ambiente virtual**

```shell
.venv\Scripts\activate
```

**Instalação do Flask**

```shell
pip install Flask
```

**Baixar atualizações caso necessário**

```shell
python.exe -m pip install --upgrade pip
```

**Executar o programa**

```shell
flask —app app run
```
